# 中国中学作业管理系统 - 前端

**⚠️ 测试版本** - 此版本为测试版本，不代表最终结果。

## 当前版本修改内容

1. 更新了登录接口调用方式
2. 添加了 `app_name` 参数支持
3. 修改了注册页面的账户类型选择（管理员/普通用户）
4. 更新了权限管理相关的 API 调用
5. 修复了 API 参数问题（`class` 改为 `class_name`）

## 文件说明

- **admin.html** - 管理端（注册账号、配置权限、修改密码）
- **teacher.html** - 教师端（创建作业、编辑学生完成情况）
- **student.html** - 学生端（查询作业情况）
