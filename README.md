# 中国中学作业管理系统 - 前端

> **⚠️ 测试版本** - 此版本为测试版本，不代表最终结果。

## 项目简介

中国中学作业管理系统前端页面，包含管理员端、教师端和学生端。

## 当前版本修改内容

1. 更新了登录接口调用方式
2. 添加了 `app_name` 参数支持
3. 修改了注册页面的账户类型选择（管理员/普通用户）
4. 更新了权限管理相关的 API 调用

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap

## 项目结构

```
├── admin.html          # 管理员端页面
├── teacher.html        # 教师端页面
├── student.html        # 学生端页面
└── README.md           # 项目说明
```

## 使用方式

直接在浏览器中打开对应的 HTML 文件，或部署到 web 服务器。
