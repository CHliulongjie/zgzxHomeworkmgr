## 修复的问题

### 1. 参数不匹配问题

**问题描述**：前端发送的参数名称与后端 API 期望的参数名称不一致

**修复内容**：

#### teacher.html
- `/subjects` 接口：参数从 `class` 改为 `class_name`
- `/assignments` 接口：参数从 `class` 改为 `class_name`
- `/students` 接口：参数从 `class` 改为 `class_name`
- `/download` 接口：参数从 `class` 改为 `class_name`

#### student.html
- `/assignments` 接口：参数从 `class` 改为 `class_name`
- `/students` 接口：参数从 `class` 改为 `class_name`

### 2. API 参数说明

后端 API 参数对照表：

| API | 参数 | 说明 |
|-----|------|------|
| `/classes` | 无 | 获取所有班级 |
| `/subjects` | class_name | 根据班级获取科目 |
| `/assignments` | class_name, subject | 获取作业列表 |
| `/students` | class_name, subject, assignment | 获取学生列表 |
| `/download` | class_name, subject | 下载 Excel 文件 |
